Code P2560 — Engine Coolant Level Low

Cooling System

This code was set because the vehicle's coolant level was low due to an active leak in the cooling system. Low coolant can cause the engine to run hotter than normal, which triggers this warning and puts the vehicle into a protective mode to prevent engine damage.

Code P130D — Engine Knock Detected

Engine / Combustion

Pending Repair

This code means the engine's computer detected abnormal combustion — commonly referred to as "knock" or "pre-ignition" — in one of the engine cylinders. When this happens, the computer tries to compensate to protect the engine, but in this case it reached its limit and stored a fault code.

Through diagnostic testing, the root cause was identified as carbon buildup on the intake valves inside the engine. This is a known and common condition on this engine type. Over time, oily deposits accumulate on the intake valves and create hot spots that cause the fuel-air mixture to ignite at the wrong moment.

This condition is separate from the coolant leak and requires its own service to fully resolve. It is not an emergency, but it should be addressed to restore full engine performance and prevent the check engine light from returning.

Summary & Recommended Service

What We Found & What's Next

Your 2013 Ford Escape came in with two diagnostic codes. The coolant leak has been fully repaired — the lower radiator hose and water control valve were replaced, and the cooling system is now in good working order.

The remaining code, related to engine knock, was thoroughly diagnosed and traced to carbon deposits on the intake valves — specifically most active in Cylinder 2. This is a maintenance issue, not a breakdown risk, but it does affect performance and will keep the check engine light on until corrected.
